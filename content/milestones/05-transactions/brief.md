# Milestone 5: Transactions

## Overview

Add ACID transaction support to your database. You'll implement begin/commit/rollback semantics, write-ahead logging for durability, and basic isolation so concurrent operations don't corrupt data.

## Learning Objectives

- ACID properties and what each one actually means in practice
- Write-ahead logging for atomicity and durability
- Concurrency control and isolation levels
